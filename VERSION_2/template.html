The Random Article Shortcode Tag is Coverted To = [Random_Article] This will pull random Spintax articles from a folder Unspin and display a different article on each refresh


The News Feed Shortcode Tag is Coverted To = [News_Feed] This displays a related newsfeed. There are many CSS Style options. Please check the CSS guide for Style Tags


The Map Shortcode Tag is Coverted To = [Map]
This will display a Google Map of the current location with NO API. The Google Map PARAMETERS can be edited within the script
